# 🌍 Overview

Weather AI Assistant is **far more than a weather application.**

It is an **Agentic AI Weather Intelligence Platform** capable of understanding natural language, reasoning over weather datasets, analyzing forecasts, comparing meteorological observations, and generating intelligent insights through multiple AI tools.

Instead of simply displaying weather information...

**It understands weather.**

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Weather_AI_Assistant.git

cd Weather_AI_Assistant
```

---

# ⚙ Backend Setup

Create Virtual Environment

```bash
python -m venv venv
```

Activate

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Start Backend

```bash
cd backend

python app.py
```

or

```bash
uvicorn app:app --reload
```

Backend runs on

```
http://localhost:8000
```

---

# 🌐 Frontend Setup

```bash
cd frontend

npm install
```

Run development server

```bash
npm run dev
```

Frontend

```
http://localhost:5173
```

---

# 🔑 Environment Variables

Create

```
backend/.env
```

Example

```env
GEMINI_API_KEY=YOUR_GEMINI_KEY

WEATHER_API_KEY=YOUR_WEATHER_API_KEY

OLLAMA_BASE_URL=http://localhost:11434

MODEL_NAME=gemini-2.5-flash
```
