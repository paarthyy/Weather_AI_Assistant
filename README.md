# 🚀 Overview

Weather AI Assistant is an **Agentic AI-powered Weather Intelligence Platform** that transforms traditional weather data into interactive insights.

Instead of merely displaying forecasts, the platform understands user queries, analyzes meteorological datasets, performs weather analytics, and visualizes results through an intelligent AI assistant.

The project combines:

- 🤖 AI Agents
- 🌍 Live Weather APIs
- 📊 Analytics Dashboard
- 📈 Forecast Visualization
- 🛰 IMD Dataset Analysis
- 🗺 Interactive Maps
- 📍 Station Explorer
- 💬 Conversational AI

# 🚀 Installation

Clone

```bash

```

Backend

```bash
cd backend

pip install -r requirements.txt

python app.py
```

Frontend

```bash
cd frontend

npm install

npm run dev
```
